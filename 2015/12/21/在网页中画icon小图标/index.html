据不完全统计，现在好多网站都使用了icon小图标，比如天猫商城、慕课网商城等。有时候使用适当的图标
能够达到一图胜千言的效果。

虽然天猫商城、慕课网商城都用了icon小图标，但其使用本质不尽相同，下面就介绍三种在网页中画icon小图标的方法： 

1、css sprite ：
也称css精灵。一项使用背景图片定位，在网页中画icon小图标的技术

2、font+html
使用font字体用html代码以文本的形式在网页中画icon小图标

3、font+css
实质与第二种方法相同，不同的是，他是将icon小图标对应的html代码结合css属性写成css样式，方便调用

一、css Sprite

在国内被称为“css雪碧”，也叫“css精灵”。它是把网页中一些零星背景图片整合到一张图片文件中，再利用css的背景图片定位到要显示的位置，因此也叫“图片拼合”技术。这样可以减少文件体积和服务器请求次数，从而提高效率。

1、知识点

1、background-image：背景图片属性，引入背景图片；

2、background-position:背景图片定位属性，通过该属性定位小图标该显示的位置

下面是利用这两个属性在网页中画icon小图标的例子：

2、对例子的阐述

.div-red i{
        height: 40px;/*图片的高度*/
        width: 24px;/*图片的宽度*/
        display: block;
        margin-left: 10px;
        margin-right: 8px;
        float: left;
        background-image: url("images/s-icon.png");/*引入背景图片*/
    }

如上所示，样式中定义了要显示的icon小图标的宽度和高度，这个宽度和高度需要和图片中所画的每个图标的宽度和高度相同。也就是说在定义样式时，需要事先规划好背景图片中每个小图标的宽度和高度。

<i style="background-position:0 -40px"></i>

这行代码用了background-positon属性定义了图标显示的位置。对于位置的定义跟图标的宽、高有关。在背景图片中以左上角为（0,0）坐标，依次根据图片宽、高往下或往右写图标的位置。

比如给出的例子中，因为每个图标的宽高都是一样的，所以可以如下这样写：

宽：24px  高：40px
background-position:0 0;/*左上角第一个图片即第一行第一个图片*/
background-position:0 -40px;/*第二行第一个图片*/

background-position:-24px 0;/*第一行第二个图片*/
background-position:-24px -40px;/*第二行第二个图片*/

按照以上所写往后一次类推，可以获取自己想要的图标。需要注意的是，定位中的值只能是负值，如果写成正值的话是不会显示图标的。

综上所述，可以得出一个background-position属性书写的基本公式：

background-position:-|横着前面图标宽度的总和|,-|竖着前面图标高度的总和|；

3、总结

1、css Sprite 画icon小图标的优点：

(1) 相对单个小图标，它节省文件体积和服务器请求次数；

(2) 一般情况下，你需要保存为PNG-24的文件格式（相对PNG-8，PNG-24没有毛边）；

(3) 可以设计出丰富多彩的颜色图标。

2、css Sprite 画icon小图标需要注意：一定要先确认icon小图标的大小，然后再根据小图标的大小来摆放小图标。

3、css Sprite 画icon小图标关键在于：用background-position相对于整张图片定位。

二、font+HTML

使用font字体用html代码以文本的形式在网页中画icon小图标。在这里给大家介绍一个下载图标的网站，有4000以上个免费图标。官方网址：https://icomoon.io

现在下来之后，我们可以发现里面的fonts文件夹里有四个不同后缀的文件，分别是.eot,.svg,.woff,.tff。这四个后缀文件的意义：

eot： Embedded OpenTypeFonts(FOT):微软开发的用于嵌入网页中的字体，IE专用字体
WOFF：The Web OpenFont Format(WOFF):Web字体中最佳格式，他是一个开发的TrueType/OpenType的压缩版本。09年被开发，如今被W3C组织推荐标准
TTF：TrueType Fonts(TTF):1980s,由Microsoft和Apple联合开发的一套字体标准，识MacOS和WIN操作系统中最常见的一种字体。
SVG：Scalable Vector Graphics(SVG):基于XML（Extensible Markup Language），由World Wide Web Consortium（W3C）联盟进行开发的。是一种图像文件格式，可缩放的矢量图形。

我们在使用的时候，要同时引入所有字体，用来支持不通浏览器。

2、知识点

在用font+html在网页中画icon小图标时，我们用到的关键点是@font-face。
@font-face是CSS3中的一个模块，他主要是把自己定义的Web字体嵌入到网页中。

1、@font-face语法

@font-face {
   font-family: <YourWebFontName>;/*设置文本的字体名称*/
   src: <source> [<format>][,<source> [<format>]];/*设置自定义字体的相对路径或者绝对路径，注意，此属性只能在@font-face规则里使用*/
   [font-weight: <weight>];/*设置文本粗细*/
   [font-style: <style>];/*设置文本样式*/
 }

2、案列

如上述代码所示，用@font-face引入字体文件，并写好样式，引入写好的样式，再将图标对应的16进制代码写入html，即可在网页中显示想要的icon小图标了。图标对应的16进制code可在下载文件中的demo.html中找到。

这里值得注意的是，在html中写入图标对应的16进制code时，需要在前面加上“&#x”，否则小图标将不会显示。

3、总结

用font+html在网页中画icon小图标的优点：

1、可根据自己的需要改变图标的颜色或其他css效果
2、可以根据font-size属性来改变图标的大小
3、任意缩放图标不会影响清晰度
4、兼容性较好，支持所有的现代浏览器

3、font+css

font+css是将小图标对应的字体写成css样式，然后引用即可。
