    一、Junit4简介

    打开Junit的官网http://junit.org，可以看到一段话：JUnit is a simple framework to write repeatable tests. It is an instance of the xUnit architecture for unit testing frameworks.
    这段话的意思是：Junit是一个用来写重复测试的简单框架。是单元测试框架Xunit的架构实例。
    也就是说，Junit是基于Xunit的一个单元测试框架。

    Xunit框架我没有深入研究，今天就介绍Junit单元测试的应用。

    二、Junit4在Java项目中的使用

    1、写一个测试用例的步骤

    (1) 新建一个Java项目，写一个类Calculate，里面写上自己的方法；

    (2) 引入Junit4的jar包：jar包可以从官网上下载，也可以用eclipse自带插件
    若用eclipse自带插件，右击项目——>properties——>Java Build Path ——> Labraries ——> Add Larbrary,
    然后选择Junit——>next——>选择Junit4，点击finish即可引入eclipse自带Junit的jar包；

    (3)新建测试类，右击类Calculate——>new ——> Junit Test Case ——> 进入新建测试用例窗口
    在此窗口中，
    source folder：要选择资源包test(没有要新建)，如我写的例子中：jUnit4Demo/test
    package: 要与Calculate所在的包结构保持一致（可以不一样，但规范性上来说是一样的）
    Name：要新建的测试用例的名字
    其他不用动，点击next——>选择需要测试的方法——>点击完成即可。
